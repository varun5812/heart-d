Heart Disease Prediction Using Machine Learning

A machine learning project that predicts the likelihood of heart disease using clinical features. Multiple machine learning models were trained and evaluated, including Logistic Regression, KNN, SVM, Naive Bayes, and Decision Tree. After comparison, the KNN model was selected for deployment in the web application.

Algorithms Used

Logistic Regression
K-Nearest Neighbors (KNN)
Support Vector Machine (SVM)
Naive Bayes
Decision Tree
Final Deployed Model
K-Nearest Neighbors (KNN) was used as the final deployed model. The application uses the saved KNN model along with the scaler and feature columns for real-time prediction.

Technologies Used

Python
Pandas
NumPy
Matplotlib

Scikit-learn

Render
Jupyter Notebook / Google Colab
Model Training and Evaluation

Data preprocessing and feature scaling using StandardScaler
Train-test split
Model comparison using Accuracy and F1 Score
Final deployment with saved KNN model artifacts
Web Deployment
The project was deployed as an interactive web application where users enter medical parameters and get an instant heart disease prediction. The app is deployed online using Render.
